Visual Basic

Visual Basic (VB) es un lenguaje de programación dirigido por eventos, desarrollado por Alan Cooper para Microsoft.Visual Basic contiene un entorno de desarrollo integrado o IDE que integra editor de textos para edición del código fuente, un depurador, un compilador (y enlazador) y un editor de interfaces gráficas o GUI.

Visual Studio 2015

Visual Studio Professional 2015 con Update 3 proporciona servicios y herramientas de desarrollo profesionales para desarrolladores individuales y pequeños equipos. Cree aplicaciones para varios dispositivos, equipos y la web, todas ellas con la tecnología de la nube y basadas en sus aplicaciones y habilidades existentes. Use las tecnologías de desarrollo multiplataforma para crear aplicaciones para Windows, Android e iOS a partir de una solución de desarrollo único y mucho más.

Novedades de Visual Studio

Visual Studio permite escribir código de manera precisa y eficiente sin perder el contexto del archivo actual. Puede acercar fácilmente los detalles, como la estructura de llamadas, las funciones relacionadas, las inserciones en el repositorio y el estado de las pruebas. También puede aprovechar nuestras funciones para refactorizar, identificar y corregir problemas de código.
